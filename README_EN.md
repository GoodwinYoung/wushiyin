# 🇯🇵 Japanese Gojuon Study

English Edition | [中文版](./README.md)

## ✨ Key Features

-   **Comprehensive Character Sets**: Includes Seion (Basic), Dakuon/Handakuon (Voiced), and Yoon (Contracted sounds).
-   **Interactive Pronunciation**: Click any card to trigger high-quality Japanese audio (powered by Web Speech API).
-   **Practice Mode (Smart Shuffle)**: 
    -   Hides Romaji hints until a character is played.
    -   Automatically **shuffles** all characters randomly every time the mode is toggled, preventing rote memorization by position.
-   **Visual-Audio Sync**: During "Speak All" mode, cards trigger a ripple effect synchronized with the audio playback.
-   **Multi-language Support (I18n)**: Seamlessly toggle between English and Chinese interfaces.
-   **Modern UI/UX**:
    -   Smooth "sliding switch" controls.
    -   Responsive grid layouts for all screen sizes.
    -   Zen-inspired color palette and micro-interactions.

## 🚀 Deployment

### 1. Local Development
```bash
npm install
npm run dev
```

### 2. Docker One-Click Deployment
The project includes an optimized `nginx.conf` with Gzip compression and SPA routing support.
```bash
# Build and start the container
docker compose up -d --build
```
Access via: `http://localhost:8080`

### 3. CI/CD Pipeline
-   **GitHub Actions**: Automatically builds and deploys to **GitHub Pages** on every push to the `main` branch.
-   **GitLab CI**: Built-in `.gitlab-ci.yml` for pushing images to private GitLab Container Registries.

## 📁 Project Structure

-   `src/composables/`: Core business logic (shuffling, audio sequence, state management).

-   `src/data/`: Complete Gojuon dataset.
